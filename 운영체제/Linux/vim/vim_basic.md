# Linux

## vim

### 저장 및 종료

 **Esc** 버튼을 누른 후, **:** 을 입력한 다음에 명령어를 입력한다.

- **:w**

  저장

- **:sav [다른 이름]**

  다른 이름으로 저장

- **:w file.txt** 

  file.txt 파일로 저장

- **:w » file.txt**

  file.txt 파일에 덧붙여서 저장

- **:q**

  vi 종료

- **:up**

  바뀐 내용만 저장합니다.

- **:x**

  :upq와 같은 내용입니다.

- **:q!**

  vi 강제 종료

- **ZZ**

  저장 후 종료

- **:e file.txt**

  file.txt파일을 불러옴

# C프로그래밍

## 자료형과 선행처리기

### 상수와 변수

#### 자료형

- 프로그램에서 사용하는 자료의 형태
- 상수와 변수로 구분해서 사용
- 프로그램에서 자료 처리를 위해서는 자료 생성, 저장 처리 과정이 필요
  - C 프로그램은 수많은 명령어와 자료들로 구성
  - 이들 자료의 정확한 처리와 효율적인 활용이 필요
- C언어는 여러 가지 종류의 자료형을 지원

##### 상수

- 항상 고정된 값을 갖는 자료
- 값을 한 번 정해지면 프로그램 도중 그 값을 변경할 수 없는 수
- 변수와 반대의 개념
- 종류
  - 정수형 상수
    - 10진수, 8진수, 16진수로 표현
  - 실수형 상수
  - 문자형 상수
  - 문자열 상수
